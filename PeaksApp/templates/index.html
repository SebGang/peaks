Mountain peaks

This site allows you to create/read/update/delete peaks, and retrieve a list of peaks in a given geographical bounding box, defined by lat min&max / long min&max.

User Guide

Through the side pannel you can acces to the list of peaks stored in the database by clicking on All peaks.
There you will be able to create/read/update/delete. To go back to home page you have to manually modify the url to http://127.0.0.1:8000/

By clicking on Filter peaks by location you will access the form to define the desired bounding box.
